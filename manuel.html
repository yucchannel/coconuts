Coconut Language Ver 3.0 Manual

Your guide to getting started with Coconut Ver 3.0

Introduction

Coconut Language Ver 3.0 is a high-level interpreted programming language designed for beginners and professionals alike. It builds upon earlier versions with added features like user-defined functions, error handling, and enhanced speed through multithreading.

Key Features

  • User-defined functions: Easily create custom functions.
  • Error handling: Catch exceptions and handle errors gracefully.
  • Multithreading: Speed up execution by parallel processing using Python's ThreadPoolExecutor.
  • Basic arithmetic and variable handling: Perform basic math operations and store variables.

Basic Commands

  • PRINT: Outputs a string or variable to the console.
  • SET: Assigns a value to a variable.
  • ADD: Adds two numbers.
  • SUB: Subtracts two numbers.

Defining Functions

You can define custom functions using the define_function command. For example: 

define_function('multiply', lambda x, y: x * y)

This allows you to call multiply with two arguments, multiplying them together.

Error Handling

Use the try_except function to handle errors. For example: 


try_except(lambda: 1 / 0, lambda e: print("Error occurred:", e))

This catches division by zero errors and prints a friendly message.

Running Coconut Code

To run Coconut code, input your commands into the interpreter. Example: 

SET x 10
PRINT x
