Introduction

Welcome to the Coconut Language manual! This guide will walk you through all the basic commands and syntax to get you started with programming in Coconut.

Basic Commands

PRINT

Used to display a message or value.

PRINT "Hello, World!"

SET

Used to create or assign a value to a variable.

SET x = 10

ADD

Used to add two variables together.

ADD x + y
