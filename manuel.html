Introduction to Coconut Language

November 2024

By: Yuto Yamamoto

Coconut is an easy-to-learn programming language designed for beginners and educators. It is built on simple commands, allowing users to get started with programming quickly. This manual provides an introduction to the core concepts of Coconut, starting from basic commands to advanced features.

With Coconut, you can learn the fundamentals of programming while building practical applications, such as simple calculators, text-based games, and more.

Basic Commands

November 2024

By: Yuto Yamamoto

Coconut provides basic commands that make it easy to write programs. Below are the most important commands that you’ll use often:

  • PRINT - Outputs text or values to the screen.
  • SET - Assigns a value to a variable.
  • ADD - Adds two numbers and stores the result.
  • IF - Allows conditional logic to control the flow of the program.

Here is a simple example of a Coconut program that adds two numbers: 


SET A TO 5
SET B TO 3
SET RESULT TO ADD A AND B
PRINT RESULT

This program sets variables A and B to 5 and 3, then adds them together and prints the result (8).

Control Structures

November 2024

By: Yuto Yamamoto

Coconut supports control structures like loops and conditionals that help you control the flow of your program. The most common ones are IF, ELSE, and WHILE.

Example of an IF statement: 


SET A TO 5
IF A IS GREATER THAN 3 THEN
    PRINT "A is greater than 3"
ELSE
    PRINT "A is not greater than 3"

In this example, the program checks if A is greater than 3. If true, it prints the first message; otherwise, it prints the second message.
